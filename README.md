THIS FILE DEFINES HOW THIS PROJECT WORKS  

Healthcheck Project:
This is a simple bash script that checks your system health and saves the results in a log file.

A.Files:

a.healthcheck.sh → main script
b.healthlog.txt → stores the output
c.README.md → explains the project

B.What it shows

a.Date and time
b.System uptime
c.CPU load
d.Memory usage
e.Disk usage
f.Top 5 memory using processes
g.Status of nginx and ssh services

C.How to run

1. Open the terminal in VS Code
2. Run this command once:
chmod +x healthcheck.sh
3. Then run the script:
./healthcheck.sh

4. Open healthlog.txt to see the output# HEALTHCHECK_PROJECT
